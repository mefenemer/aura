Trust & Security

Last updated: 10 June 2026 · Reviewed quarterly

GDPR Compliance: Be More Swan acts as a Data Processor under UK GDPR / EU GDPR. Our full Privacy Policy is available at /privacy.html. For erasure requests, please email hello@bemoreswan.com.

Data Residency

All customer data processed and stored by Be More Swan resides within the United Kingdom / European Economic Area on infrastructure provided by Netlify (Cloudflare CDN with UK/EU edge nodes) and our managed PostgreSQL provider. No data is transferred outside the UK/EEA without adequate safeguards as required by UK GDPR Article 46.

Encryption Standards

  • ✓In transit: All data is encrypted in transit using TLS 1.3. Older cipher suites are disabled.
  • ✓At rest: All database storage is encrypted at rest using AES-256 via the managed database provider's native encryption.
  • ✓OAuth credentials & API keys: Stored with AES-256-GCM application-layer encryption. Plaintext credentials are never logged or returned to clients.

Penetration Testing

Be More Swan undergoes regular security assessments. Our last penetration test was conducted in Q2 2026. A summary report is available on request to enterprise customers — please email hello@bemoreswan.com with the subject "Security Report Request".

Data Retention

  • ✓Active accounts: Data is retained for the duration of your subscription.
  • ✓After account deletion: Personal data is permanently deleted within 30 days of account closure. An anonymised erasure record is retained for compliance purposes.
  • ✓Audit logs: Retained for 12 months for security and compliance monitoring, then purged.
  • ✓Integration API call logs: Retained for 90 days, then automatically purged.
